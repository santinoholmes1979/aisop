# AISOP

**AISOP (AI Security Operations Platform)** is a defense-style AI SOC platform prototype designed to demonstrate end-to-end security operations workflows.

The platform is being built to unify:
- telemetry ingestion
- detection engineering
- behavioral analytics
- risk scoring
- incident correlation
- AI-assisted triage
- analyst dashboards
- case management

## Phase 1 Status
Project skeleton and Streamlit application shell in progress.

## Planned Modules
- Overview
- Alerts
- Investigations
- AI Triage
- Hunting
- Metrics
